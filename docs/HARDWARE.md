## Reproducibility

To reproduce the project:

1. Clone the GitHub repository.
2. Install dependencies using npm.
3. Run the Next.js application locally.
4. Launch the Aegis Mesh AI dashboard.
5. Test AI agents and QVAC inference workflow.


## Privacy & Security

- No external AI APIs are used.
- All AI processing is designed for local execution.
- User data is not sent to cloud AI providers.
- Mesh communication is designed for private device collaboration.
